## 저작권 문제에 대해서

모바일웹 강좌의 텀 프로젝트로 개발한 것으로 **절대 상업적으로 사용하지 않는다.**

- TOPIC NEWS는 어떤 기사도 수집하지 않는다.
- TOPIC NEWS는 기사의 본문을 변경하거나 사용하지 않는다.
- TOPIC NEWS는 모든 기사에 대해서 제목과 직접링크만 제공한다.
